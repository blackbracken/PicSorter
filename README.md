# PicSorter
*An Android app for assisting to sort pictures*

# 概要
画像管理を補助するAndroidアプリケーションです.

# 技術
* MVVM + DataBinding
* Kotlin Coroutine + Flow
* Navigation
* Koin
* Coil
* Groupie

# インストール
[GooglePlay](https://play.google.com/store/apps/details?id=black.bracken.picsorter)

# 環境
* Android 8+ (SDK 26+)
* Kotlin 1.3.41
